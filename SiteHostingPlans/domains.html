Domain

In simple words, domain name is a system of identifying title given to a system of computers. Domain Name is the most recognized system for assigning addresses to Internet web servers. A domain name can be at most of sixty-three characters and a minimum of 1 character. Domain names are assigned to identify one or more IP addresses. Any name registered in the Domain Name System is a domain name. Basically, a domain name represents an IP resource, like a computer used to access the Internet world, a server computer hosting a website. The first level set of domain names are named as the top-level domains often represented as TLDs. Below the top level domains in the DNS hierarchy are the second-level and third-level domain names that are basically open for reservation by end-users who wish to connect the local area networks to the Internet, make other publicly accessible Internet sources or run websites. However, the registration of these domain names is usually mastered by domain name registrars who sell their services to the public. Domain names are generally written in small letters, but labels in the Domain Name System are not case sensitive. Each domain name has a suffix that indicates to which top level domain it belongs.
